L7mp is a multiprotocol service proxy and a service mesh framework. L7mp can handle essentially any transport- and application-layer network protocol natively and transparently convert between different protocol encapsulations, exposing a simple high-level Kubernetes API that allows to maintain versatile load-balancing, resilience, security and monitoring policies in a protocol-agnostic manner.


The intention is for l7mp to serve as an incubator project to prototype the main service mesh features that are indispensable to support network-intensive legacy/non-HTTP applications seamlessly in Kubernetes.
